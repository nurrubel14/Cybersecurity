Typosquatting
Typoquatting is a domain attack that targets users who input a website address incorrectly into a web browser instead of using a search engine. For instance, instead of inputting www.pbebank.com, a user may enter bebankk.com, which will lead them to a fake website without their knowledge.

The attackers will next attempt to mimic the appearance and feel of the legitimate websites so that visitors believe it is authentic and feel safe providing sensitive information.

Attackers launch typosquatting attacks by purchasing and activating a domain name that is a misspelling of a well-known website. The process generally has the following sequence:

Choose the target website: Attackers first choose which website to impersonate. In general, they seek popular websites with high traffic so they can reap the benefits quickly.
Impersonate the authorized domain: The attackers must register the typosquatting URL to impersonate the authorized domain. Attackers frequently register several URLs.
Mislead users: Ultimately, the attackers will trick users into accessing their fake domains.
